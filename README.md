# Love, Synthetic
AI 情感情书生成与情感轨迹可视化产品

## 项目背景
在日常情感表达场景中，用户普遍缺乏层次化、递进式的表达能力。
市面上多数 AI 写作工具仅提供单次文本生成，缺少情感递进与可视化呈现。
本项目旨在探索 AI 在情感交互场景的产品化潜力。

## 项目介绍
独立完成的端到端 AI 产品项目，覆盖需求分析、产品设计、模型微调、数据处理、可视化与部署全流程。
项目实现 30 天递进式情感情书生成，并通过文本嵌入与降维技术完成情感轨迹可视化。

## 核心工作
- 独立完成产品定位、需求分析与功能设计
- 构建自定义情感语料，使用 Prompt 策略微调 GPT-2 模型
- 运用 Sentence-BERT 与 t-SNE 实现情感编码与可视化
- 设计情感轨迹可视化方案，提升用户情感感知体验
- 完成 GitHub Pages 部署，验证产品化可行性
- 总结 AI 情感内容生成的设计模式与落地方法

## 技术栈
Python、GPT-2、Sentence-BERT、t-SNE、数据可视化、GitHub Pages

## 项目价值
验证了 AI 模拟递进式情感的可行性，可应用于情感交互产品、内容创作工具、个性化表达与文创互动产品方向。
