# Love, Synthetic
AI Emotional Love Letter Generation & Visualization Product

## Project Background
Users often lack progressive and layered emotional expression in daily communication. Most AI text generation tools only provide single-round output without emotional progression or visual presentation. This project explores the productization potential of AI in emotional interaction.

## Project Overview
Independent end-to-end AI product project. I completed the full process: requirement analysis, product design, model fine-tuning, data processing, visualization, and deployment.

The project generates 30-day progressive emotional love letters and visualizes emotional trajectory changes through embedding and dimensionality reduction.

## Core Implementation
- Independently designed product positioning and core functions
- Built custom emotional corpus and fine-tuned GPT-2 with Prompt-Based method
- Applied Sentence-BERT for text embedding and t-SNE for dimensionality reduction
- Implemented emotional trajectory visualization to enhance user experience
- Deployed the project on GitHub Pages and verified product feasibility
- Summarized design patterns for AI-generated emotional content

## Tech Stack
Python, GPT-2, Sentence-BERT, t-SNE, Data Visualization, GitHub Pages

## Project Value
This project proves the feasibility of AI-simulated progressive emotion. It can be applied to emotional interaction products, content creation tools, personalized expression, and interactive creative products.
